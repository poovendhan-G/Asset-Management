 Asset Management System

This is a simple Asset Management System developed using Node.js and PostgreSQL.
The application helps to manage employees and assets, and track asset usage.


1. Employee Module

* Add employee
* View employee list

2. Asset Module

* Add asset
* View assets

3. Asset Allocation

* Issue asset to employee
* Return asset

4. Asset History

* View asset issue and return history


To Run

1. Install dependencies

```
npm install
```

2. Start server

```
npx nodemon app.js
```

3. Open in browser

```
http://localhost:3000
```


Sample URLs

* View Employees:
  http://localhost:3000/employees

* Employee Page (UI):
  http://localhost:3000/employees/page

* View Assets:
  http://localhost:3000/assets

* Asset History:
  http://localhost:3000/assets/history/1
